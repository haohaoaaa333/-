### 2026-07-02 (晚间续4) — 学习页网格布局修复

- 用户截图反馈学习页科目卡片没有居中，且宽度和“资料分析”特色卡片一样长，是单列而非两列。
- 原因：`.subject-card` 使用了 `width: calc(50% - 8rpx)`，同时有 `padding` 和 `border`。在小程序样式中默认是 `content-box`，padding 和 border 会额外加到宽度上，导致实际宽度超过 50%，两张卡片放不下，就被迫独占一行。
- 修复：给 `.subject-card` 和 `.featured-card` 都加上 `box-sizing: border-box`，让 padding 和 border 被包含在 width 内。同时给 `.subjects-grid` 加了 `justify-content: space-between` 保证两行之间排布整齐。
